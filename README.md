Student Contact Manager
This is my Student Contact Manager project made using Python Flask and SQLite database.
 
In this project we can:
-Add student details
-View all students
-Search students
-Edit student information
-Delete student records
This project is based on CRUD operations.

Technologies Used:
.Python
.Flask
.SQLite
.HTML
.CSS

Installation & Setup:
1.Clone the repository
https://github.com/amishra17-cyber/student-contact-manager.git
2.Open project folder
cd student-contact-manager
3.Install Flask
pip install flask
4.Run the application
python app.py
5.Open in Browser
http://127.0.0.1:5000/

Database
This project uses SQLite database.
Database file: database.db

Table used: students
